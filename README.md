# VoltGuard FieldLink Backend

VoltGuard FieldLink is a web-based electrical asset testing and reporting backend.

## Projects

- VoltGuard.Api
- VoltGuard.Application
- VoltGuard.Domain
- VoltGuard.Infrastructure
- VoltGuard.Shared
- VoltGuard.UnitTests
- VoltGuard.IntegrationTests

## Run

dotnet restore
dotnet build
dotnet run --project src/VoltGuard.Api
